# Dark

A proof of concept of an Electron-like Deno app for Windows 11

## Installing

Install Deno with powershell: `irm https://deno.land/install.ps1 | iex`

## Running

`deno task dev`

## Building for Windows 11

`deno task build`

## Known issues

- Debug toggle doesn't work in Webview
